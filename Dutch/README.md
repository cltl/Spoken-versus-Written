# Reproducing our results

This file contains all instructions to reproduce our main results.

## Requirements

* Python 3.6
    - tabulate 0.7.7
    - spacy 2.0.4
    - pyphen 0.9.4
    - matplotlib
    - numpy 1.13.1
    - nltk 3.2.2
    - seaborn 0.7.1

## Reproducing the main results

Run the following commands in order:

* `bash run_analysis.sh`
* `python generate_table.py`
* `cd PID_rater`
* `python PID_rater.py`
